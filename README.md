# Deloitte-data-analytics-simulation
Virtual internship project: Data analysis and dashboard creation from Deloitte Forage simulation
 Problem Statement
Simulated a Deloitte client scenario requiring data classification, analysis, and visualization to support business decision-making.

# Tools Used
- Microsoft Excel
- Tableau
- Data Cleaning
- Reporting and Dashboarding

# What Was Done
- Cleaned and structured client data in Excel
- Classified data and aggregated key metrics
- Created a Tableau dashboard to visualize trends and anomalies
- Derived actionable business insights from data patterns

# Key Insights
- Identified key data trends and patterns affecting client operations
- Highlighted areas requiring process improvements
- Recommended data-driven decisions for operational efficiency

# Business Value
Demonstrates ability to analyze complex datasets, create visual dashboards, and provide actionable insights in a professional context
